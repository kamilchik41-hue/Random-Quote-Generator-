## Описание

Random Quote Generator - GUI-приложение на Python с использованием `tkinter`.
Программа выбирает случайную цитату из списка предопределенных цитат,
показывает историю генераций, позволяет фильтровать историю по автору и теме,
а также сохраняет историю в JSON-файл `quote_history.json`.

## Возможности

- генерация случайной цитаты кнопкой `Generate quote`;
- отображение автора и темы цитаты;
- история сгенерированных цитат;
- фильтрация истории по автору;
- фильтрация истории по теме;
- добавление новой цитаты через форму;
- проверка пустых строк при добавлении цитаты;
- сохранение и загрузка истории в формате JSON.

## Запуск

Требуется Python 3. Tkinter обычно входит в стандартную установку Python.

```bash
python quote_generator.py
```

## Примеры использования

1. Запустите приложение.
2. Нажмите `Generate quote`.
3. Проверьте, что цитата появилась в блоке текущей цитаты и в истории.
4. Выберите автора в фильтре `Author`, чтобы увидеть только его цитаты.
5. Выберите тему в фильтре `Topic`, чтобы увидеть цитаты выбранной темы.
6. Добавьте новую цитату через форму `Add a new quote`.
7. Попробуйте оставить одно из полей пустым - приложение покажет предупреждение.
8. Нажмите `Save history`, закройте приложение и запустите его снова.
   История должна загрузиться из `quote_history.json`.
